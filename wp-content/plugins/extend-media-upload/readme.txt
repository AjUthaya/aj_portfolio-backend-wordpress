== Description ==

= Add folder specification and date time specification to the media uploader. =
* Can upload media to the specified folder.
* Can upload media to the specified date time.
* If you want to organize media library by folders, Please use the [Organize Media Library by Folders](https://wordpress.org/plugins/organize-media-library/).

[youtube https://www.youtube.com/watch?v=czy5KA_7xHY]

== Installation ==

1. Upload `extend-media-upload` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
